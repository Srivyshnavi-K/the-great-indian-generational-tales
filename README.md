# The Great Indian Generational Tales

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy (GitHub Pages)

1. Push changes to the `main` branch – the `Deploy to GitHub Pages` workflow builds the app and publishes the `dist` folder.
2. On the first deploy, open **Settings ▸ Pages** in GitHub and set **Build and deployment** → **Source** to **GitHub Actions**.
